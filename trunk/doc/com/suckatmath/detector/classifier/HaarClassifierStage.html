Packagecom.suckatmath.detector.classifier
Classpublic class HaarClassifierStage

Second level of a HaarClassifier. A stage is composed of several trees. The output of the trees is summed, compared against a threshold and either true or false is returned. Also has parent and next because the xml file declares them, but they are unused as of now.
