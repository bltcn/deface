Classpublic class IntegralImage

Both integral image and squared integral image + utility functions and properties. Integral image is described in Viola and Jones. It's basically an array corresponding to pixels in which each value is the sum of all pixels up and to the left of the array position, inclusive. Squared integral image is similar, but with squared pixel values Also has a tilted sum array, in which the values are the sum of all pixels in the image above and to the left if the image were rotated 45 degrees.
